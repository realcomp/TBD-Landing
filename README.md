# TBD Landing

Это проект лендинга, созданный с использованием современных веб-технологий.

## Быстрый старт

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите проект:
   ```bash
   npm run dev
   ```

- [**Архитектура проекта**](./docs/architecture.md) — обзор используемых технологий и структуры папок.

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
