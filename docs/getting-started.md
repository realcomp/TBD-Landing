# Руководство по запуску проекта

Для работы с проектом вам понадобится установленный [Node.js](https://nodejs.org/) (рекомендуется версия 18 или выше) и менеджер пакетов (npm, yarn или bun).

## Установка зависимостей

Перед первым запуском необходимо установить все зависимости:

```bash
npm install
```

Или, если вы используете другие менеджеры пакетов:

```bash
yarn install
# или
bun install
```

## Запуск в режиме разработки

Для запуска локального сервера разработки выполните:

```bash
npm run dev
```

После этого проект будет доступен по адресу [http://localhost:8080](http://localhost:8080) (или другой порт, указанный в консоли). В этом режиме поддерживается горячая перезагрузка (HMR) — изменения в коде сразу отображаются в браузере.

## Сборка для продакшена

Для создания оптимизированной сборки проекта выполните:

```bash
npm run build
```

Результат сборки будет находиться в директории `dist`.

## Просмотр собранного проекта

Чтобы проверить, как работает собранная версия проекта локально:

```bash
npm run preview
```

## Другие команды

- `npm run lint` — запуск проверки кода линтером (ESLint).
