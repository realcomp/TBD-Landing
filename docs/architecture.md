# Архитектура проекта

Проект построен на базе **Vite + React + TypeScript** с использованием современных инструментов для стилизации и компонентов.

## Технологический стек

- **Vite** — высокопроизводительный инструмент сборки.
- **React** — библиотека для создания пользовательских интерфейсов.
- **TypeScript** — строгая типизация для повышения надежности кода.
- **Tailwind CSS** — утилитарный CSS-фреймворк для быстрой стилизации.
- **shadcn/ui** — библиотека переиспользуемых компонентов на базе Radix UI.
- **Lucide React** — набор иконок.
- **TanStack Query** (React Query) — управление состоянием данных и запросами.

## Структура папок

- `src/` — исходный код приложения.
  - `components/` — React-компоненты.
    - `ui/` — базовые компоненты из shadcn/ui.
    - `sections/` — крупные блоки/секции лендинга (например, Hero, Features, UseCases).
  - `hooks/` — кастомные React-хуки.
  - `lib/` — вспомогательные утилиты и конфигурации (например, настройки `utils.ts` для Tailwind).
  - `pages/` — основные страницы приложения (если их несколько).
  - `index.css` — глобальные стили и конфигурация Tailwind.
  - `App.tsx` — корневой компонент приложения.
  - `main.tsx` — точка входа.
- `public/` — статические ресурсы (изображения, шрифты, favicon).
- `docs/` — документация проекта.
- `tailwind.config.ts` — конфигурация Tailwind CSS.
- `tsconfig.json` — конфигурация TypeScript.
- `vite.config.ts` — конфигурация сборки Vite.
