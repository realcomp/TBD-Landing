# Функциональные возможности лендинга

Данный лендинг предназначен для демонстрации возможностей продукта (предположительно таск-менеджера или системы продуктивности).

## Основные секции

1. **Hero Section** — главный экран с заголовком, призывом к действию (CTA) и фоновым изображением.
2. **Use Cases (Кейсы использования)** — описание сценариев, в которых продукт будет полезен (например: разработчики, люди с семьей, фрилансеры).
3. **Method (Метод)** — описание методологии, лежащей в основе продукта (например, GTD — Getting Things Done).
4. **Pain Points (Проблемы)** — секция, в которой пользователи могут узнать себя и свои трудности, которые решает продукт.
5. **Features (Возможности)** — детальный разбор функций приложения.
6. **FAQ** — ответы на часто задаваемые вопросы.

## Компоненты UI

В проекте используется `shadcn/ui`, что обеспечивает:
- Доступность (A11y).
- Адаптивность под мобильные устройства.
- Консистентный дизайн через CSS-переменные.
- Поддержку темной и светлой тем (через `next-themes`).
